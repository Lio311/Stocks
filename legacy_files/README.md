# מעקב מניות - Stock Portfolio Tracker

מעקב אחר תיק מניות אישי עם ויזואליזציה וייבוא מ-Excel.

## תכונות

- 📊 מעקב תיק מניות
- 📁 **טעינה אוטומטית מ-Excel** - הקובץ `תיק מניות.xlsx` נטען אוטומטית!
- 📤 ייבוא קובץ Excel אחר
- ➕ הוספת מניות ידנית
- 💰 חישוב רווח/הפסד אוטומטי
- 📈 גרף התפלגות תיק
- 💾 שמירה מקומית (LocalStorage)

## איך זה עובד

### טעינה אוטומטית

כשהאתר נפתח, הוא **אוטומטית** מחפש ונטען את הקובץ `תיק מניות.xlsx` מתיקיית הפרויקט.
- אם הקובץ קיים - התיק נטען מיד
- אם הקובץ לא קיים - מתחילים עם תיק ריק

### אופציה 1: ייבוא מ-Excel אחר

1. הכן קובץ Excel עם העמודות הבאות:
   - **סימול** (או Ticker/Symbol)
   - **כמות** (או Shares/Quantity)
   - **מחיר רכישה** (או Buy Price/Price/Cost)

2. לחץ על "העלה קובץ Excel"
3. בחר את הקובץ שלך (.xlsx או .xls)
4. לחץ "ייבא מ-Excel"
5. זהו! התיק שלך יוצג מיד

**דוגמה לקובץ Excel:**
```
סימול    כמות    מחיר רכישה
AAPL     10      150.00
MSFT     5       280.00
GOOGL    3       2800.00
```

### אופציה 2: הוספה ידנית

1. הזן סימול מניה (למשל: AAPL)
2. הזן כמות מניות
3. הזן מחיר רכישה
4. לחץ "הוסף לתיק"

## טכנולוגיות

- **SheetJS (xlsx)** - קריאת קבצי Excel
- **Chart.js** - גרפים אינטראקטיביים
- **LocalStorage** - שמירת נתונים
